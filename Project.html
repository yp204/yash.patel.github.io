I used Python's built-in functions for input handling and string manipulation, along with libraries like random to select words dynamically from a predefined list. WireShark Analysis

In my Kali VM, I utilized Wireshark to analyze OT traffic and specifically examine Modbus communication. In the Packet Detail View, I delved into Modbus packet structures, inspecting fields like function codes, transaction identifiers, and data payloads for detailed analysis. I conducted focused analyses on various Modbus functions, such as Function Code 01 (Read Coils) and Function Code 03 (Read Holding Registers), identifying their purpose and behavior within the traffic. Using TShark, I performed command-line Modbus traffic analysis, extracting relevant field parameters to isolate transaction-specific data. This included using -T fields and -e options to extract Modbus data fields effectively. Additionally, I exported Modbus traffic data into CSV format using TShark, enabling structured offline analysis. Finally, I applied Wireshark filters to isolate packets based on specific IP addresses and Modbus function codes, streamlining the investigation process.
